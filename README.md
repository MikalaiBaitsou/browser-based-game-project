# HANGMAN
                                                            



📝 Description
---------------------------------------------------------------------------------------------------------------------------
A hangman game where the player must guess a words from a hint provided and type that word out using an on-screen keyboard.  

🎯 How to Win
---------------------------------------------------------------------------------------------------------------------------
Guess the word correctly from the hint, you have 6 tries.
